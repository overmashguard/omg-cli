# omg-cli

omg-cli — это инструмент командной строки, который устанавливает защищенный туннель с использованием сети Yggdrasil. Он позволяет пользователям устанавливать безопасные соединения между системами с использованием сквозного шифрования через сетку IPv6.

## Установка

Чтобы установить omg-cli, выполните следующую команду в вашем терминале:

```bash
curl -fsSL https://raw.githubusercontent.com/your-username/omg-cli/main/install.sh | bash
```

Это загрузит и выполнит скрипт установки, который:
- Проверит вашу систему Linux и установит необходимые зависимости
- Установит и настроит сеть Yggdrasil
- Настроит правила брандмауэра для разрешения только SSH-соединений
- Настроит доступ по SSH через сеть Yggdrasil
- Создаст или настроит учетную запись пользователя для безопасного доступа

## Особенности

- **Безопасный туннель**: Создает защищенное соединение с использованием сети Yggdrasil
- **Автоматическая настройка брандмауэра**: Настраивает iptables, nftables или ufw для разрешения только SSH-соединений
- **Гибкое управление пользователями**: Позволяет создать нового пользователя или использовать существующего
- **Генерация случайных учетных данных**: Генерирует случайные имена пользователей и пароли для безопасности
- **Сетка IPv6**: Использует возможности сетки IPv6 Yggdrasil

## Использование

После установки скрипт предоставит вам:
- Адрес сети Yggdrasil
- Номер порта SSH
- Имя пользователя
- Пароль

Затем вы можете подключиться к безопасному туннелю, используя:
```bash
ssh <имя_пользователя>@<адрес_yggdrasil> -p <порт>
```

## Требования

- Операционная система на базе Linux
- Доступ root/sudo
- Подключение к Интернету
- curl или wget

## Безопасность

- Через брандмауэр разрешены только SSH-соединения
- Генерация надежных паролей
- Сквозное шифрование через сеть Yggdrasil
- Изолированный сетевой доступ
